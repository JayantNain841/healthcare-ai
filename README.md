# AI Healthcare Diagnosis System

## Overview
This project is an AI-powered healthcare assistant that predicts possible diseases based on user symptoms.

## Features
- AI Chat Assistant (LLM using Ollama)
- Symptom-based Disease Prediction
- React Frontend
- Node.js Backend
- Docker Containerization
- CI/CD Pipeline using GitHub Actions

## Technologies Used
- React
- Node.js
- Express
- Docker
- GitHub Actions
- Ollama LLM

## How to Run
docker compose up --build
